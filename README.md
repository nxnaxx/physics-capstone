## 기술 스택

[![HardWare](https://skillicons.dev/icons?i=raspberrypi,ubuntu,ros,bootstrap,js,python,nodejs,mysql,opencv)](https://skillicons.dev)

### 주요 H/W
- **Raspberry Pi**: 로봇 제어 및 데이터 처리
- **LIDAR**: 거리 측정 및 매핑
- **Camera Module**: 바코드 및 QR 코드 인식

### OS
- **Ubuntu**: Raspberry Pi 운영체제

### Mapping, Robot Control
- **ROS (Robot Operating System)**: LIDAR 기반 SLAM 매핑 및 객체 추적

### FrontEnd
- **Bootstrap**: UI 디자인 및 레이아웃 구현
- **JavaScript**: 실시간 데이터 통신(WebSocket)

### BackEnd
- **Python**: LIDAR 데이터 처리, SLAM 구현, 바코드 및 QR 코드 인식
- **Node.js**: 실시간 데이터 서버 및 API 처리
- **MySQL**: 상품 정보 DB
